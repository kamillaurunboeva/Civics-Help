# Civics Help

## Overview
iOS app that helps users prepare for the U.S. Citizenship test.

## Features
- Bilingual (English/Russian)
- Quiz mode
- JSON-based data storage
- Timer functionality

## Tech Stack
Swift, SwiftUI, Xcode
